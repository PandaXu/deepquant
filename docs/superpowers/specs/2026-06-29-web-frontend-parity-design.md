# DeepQuant Web 前端功能对齐设计

**日期:** 2026-06-29  
**目标:** Web 前端功能与桌面 GUI 完全一致  
**技术栈:** Vue 3 CDN + ECharts + 纯 CSS 暗色主题

## 架构

单页应用（`index.html`），5 个 Tab 切换：行情交易、数据管理、策略、日志、设置。  
`apps.html` 保留不变（子应用独立页面）。  
状态管理通过 `store.js` (WebSocket + REST API) 共享。

## Tab 1: 行情交易

### 布局（4 区域）
- **区域 1 (左上):** K线图 + 深度图合并。ECharts 蜡烛图（上）+ 成交量（下），下方叠加五档盘口
- **区域 2 (右上):** 下单面板。合约三级选择、方向/开平/类型/价格/数量、买入(红)/卖出(绿)、全撤、一键全平
- **区域 3 (全宽):** 行情条，水平滚动
- **区域 4 (下半部):** 持仓表 + 活动订单表 + 成交表 + 账户摘要（标签切换）

### 新增功能（15 项）
1. K线图 — ECharts candlestick + 成交量 + zoom/tooltip/crosshair
2. 选择合约自动加载历史K线 + 实时tick合并更新
3. 五档盘口 — 价格点击填价
4. 价格自动更新勾选框
5. 到期合约过滤 + 过期标识
6. 网关选择器（API 加载）
7. 一键全平按钮
8. 持仓快捷平仓按钮（每行）
9. 持仓新增列：昨仓、冻结
10. PnL 红盈绿亏
11. 活动/全部订单切换
12. 列排序（所有表格）
13. CSV 导出（所有表格）
14. 账户完整多列表格
15. 持仓颜色：涨红跌绿

## Tab 2: 数据管理

### 内容
- 合约查询表格（13列，搜索/筛选/CSV导出），对标桌面 ContractManager
- 历史数据下载（三级联动 + 周期 + 日期范围 + 进度条）
- 已导入数据概览表格

### 新增功能（3 项）
1. 合约查询 13 列表格
2. 下载进度条
3. 数据概览表格优化

## Tab 3: 策略

### 内容
- CTA 策略管理 — 策略卡片（Vue 组件渲染，非原生 DOM），含生命周期按钮、参数/变量展开、实时日志
- 策略回测 — 配置表单 + 权益曲线图 + 结果统计

### 改进功能（4 项）
1. 策略卡片改为 Vue template 渲染
2. 策略日志 WebSocket 推送（替代 3 秒轮询）
3. 回测结果 localStorage 持久化
4. 回测权益曲线图（ECharts）

## Tab 4: 日志

### 内容
- 日志表格（毫秒时间、级别、来源、消息）
- 自动滚动（上滚暂停）、级别筛选、搜索
- CSV 导出、上限 1000 条

### 新增功能（全部新建，4 项）
1. 日志表格组件
2. 级别筛选 + 搜索
3. 自动滚动控制
4. CSV 导出

- 账户表格（完整多列）
- 全局配置（字体/字号/主题/交易所可见性）

## 技术约束

- 无构建工具（Vue 3 CDN + ECharts CDN）
- 暗色主题，红涨绿跌（中国期货习惯）
- WebSocket 实时通信（复用 `store.js`）
- REST API 查询（复用 `$apiGet`/`$apiPost`）
- 响应式：900px 以下单列布局
